Node.js - open-doc
===================

Quickly open url of npm package document in browser.


Installation
-------------

```
npm install -g open-doc
```

Usage
------

```
open-doc gulp
```


Similar package
---------------

If you prefer to view document in terminal, you could use [nd](https://www.npmjs.com/package/nd) instead.
